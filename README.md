# HighSo WordPress Theme

Modern WordPress e-commerce theme inspired by the HIGH SO fashion store design at highssoo.com. This theme combines WooCommerce support with blog/magazine capabilities for a comprehensive online presence.

## Features

### E-Commerce Features
- **Full WooCommerce Integration** - Complete product catalog support
- **Product Grid Layouts** - Beautiful responsive product displays
- **Discount Badges** - Show sale percentages and special offers
- **Size Variants** - Display product size options
- **Promo Code Banners** - Prominent discount code displays
- **Mobile Bottom Navigation** - Quick access to shop, wishlist, cart, account
- **Wishlist & Cart Icons** - With live counters

### Blog & Magazine Features
- **Blog Grid Layout** - Modern card-based article displays
- **Featured Images** - Large, attractive post thumbnails
- **Category Archives** - Organized content browsing
- **Responsive Typography** - Optimized reading experience
- **Social Sharing** - Built-in sharing capabilities

### Design Elements
- **Hero Sections** - Full-width promotional banners
- **Category Cards** - Visual category navigation
- **Sticky Header** - Always-accessible navigation
- **Mobile Responsive** - Optimized for all screen sizes
- **Custom Widgets** - Flexible sidebar and footer areas
- **Newsletter Integration** - Email subscription forms

## Customization

### Colors
The theme uses a minimal black and white color scheme with accent colors for sales/discounts:
- Primary: #000000 (Black)
- Background: #FFFFFF (White)
- Accent: #ff5722 (Sale/Discount orange-red)

### Typography
System font stack for optimal performance:
```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, sans-serif;
```
